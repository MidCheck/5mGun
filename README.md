# 5mGun — 5 分钟爽枪

> 一款在浏览器里即点即玩的多人第一人称射击（FPS）游戏，专为白领碎片时间设计：
> 拉上三五好友，5–10 分钟一局，强爽点、快节奏、释放压力。

## 是什么

- **纯网页**：点链接即玩，无需下载安装。PC 键鼠、移动端虚拟摇杆均可。
- **两种模式**
  - **PvP**：团队歼灭 / 爆头模式，小队对枪，快进快出。
  - **PvE**：组队刷丧尸，波次推进，越打越爽、越打越强。
- **强爽点**：CF 风格的爆头特效与音效（版权安全的原创/CC 素材）、连杀播报、丧尸分级掉落、局内即时成长。

## 技术栈

| 层 | 选型 |
|---|---|
| 渲染 | Three.js (WebGL) |
| 客户端框架 | Vite + TypeScript |
| 网络 | WebSocket，权威服务器 |
| 房间/状态同步 | Colyseus (Node.js) |
| 持久化 | SQLite（原型）→ PostgreSQL（上线） |
| 部署 | 单云主机起步 → 房间编排弹性扩容 |

## 快速开始

```bash
npm install
npm run dev          # 同时起服务器(:2567) + 客户端(:5173)
```
浏览器打开 **http://localhost:5173** → 输入昵称 → 选模式 → **单人快速开打**（或创建房间发码联机）。
完整运行/部署/操作见 **[RUNNING.md](RUNNING.md)**。

## 工程结构

```
shared/   前后端共享的数值配置(武器/丧尸/缩放/升级)、类型、地图、数学
server/   Node + Colyseus 权威服务器（房间/模拟/命中/AI/波次/人机）
client/   Vite + Three.js 客户端（渲染/预测/输入/音频/特效/HUD/UI）
docs/     设计与技术文档
```
